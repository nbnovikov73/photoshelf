# Design Language

Keywords: editorial, noir, restrained, architectural, tactile, monochrome-friendly, quiet premium.

References: Richard Avedon for clarity and portrait intensity; Helmut Newton for tension, geometry and power; Peter Lindbergh for black-and-white naturalism; Japanese urban noir for darkness, reflections, concrete and neon; industrial architecture for structure and proportion.

UI direction: large photography, minimal chrome, strong typography, generous margins, calm transitions, no decorative clutter.
